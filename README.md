# BASA Radiation Guide

Production-ready, mobile-first React app for radiation oncology patient education.

## Tech Stack

- React + TypeScript
- Tailwind CSS
- React Router
- Context API + localStorage persistence

## Features

- Onboarding flow (language + treatment site)
- Global patient profile context for future AI integrations
- Responsive navigation (top desktop, bottom mobile)
- Chat interface with dynamic treatment-specific starter questions
- Radiation guide, searchable FAQ, contact, and settings pages
- Safety notice, medical disclaimer, and emergency guidance
- English, Telugu, and Hindi localization support

## Commands

```bash
npm install
npm run dev
npm run lint
npm run build
```
